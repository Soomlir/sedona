# sedona

[Опубликованная версия](https://soomlir.github.io/sedona/).

## Инструкция по применению

### Установка модулей

```bash
npm ci
```

### Сборка проекта в папку build

```bash
npm run build
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Запуск линтеров и валидаторов

```bash
npm run lint
```
